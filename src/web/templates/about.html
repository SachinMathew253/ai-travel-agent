üöÄ What is AI Travel Planner?

AI Travel Planner is a cutting-edge multi-agent system that leverages Google's Agent Development Kit (ADK) and the Model Context Protocol (MCP) to create comprehensive, personalized travel plans. Our system employs multiple specialized AI agents that work together to handle different aspects of travel planning.

Key Innovation: Instead of using a single AI to handle everything, we deploy specialized agents that excel in their specific domains - flights, hotels, activities, and weather - then coordinate them intelligently to create superior travel experiences.

üõ†Ô∏è Technical Implementation

MCP Server Architecture

Each service (weather, flights, hotels, activities) runs as an independent MCP server:

# Example: Weather MCP Server from fastmcp import FastMCP mcp = FastMCP("Weather Server", host="localhost", port=8000) @mcp.tool() def get_weather(city: str) -> str: """Get current weather for a city""" # Weather API integration return weather_data @mcp.tool() def get_forecast(city: str, days: int) -> str: """Get multi-day weather forecast""" # Forecast API integration return forecast_data

ADK Agent Integration

Google ADK orchestrates the MCP servers through specialized agents:

# Example: Creating specialized agents from google.adk.agents import LlmAgent from google.adk.tools.mcp_tool import MCPToolset weather_toolset = MCPToolset({"url": "http://localhost:8000"}) weather_agent = LlmAgent( name="Weather Agent", instructions="Provide weather info and travel advice", tools=[weather_toolset], model="gemini-2.0-flash-exp" )
